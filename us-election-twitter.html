U.S. Election Discourse on Twitter

Demo / Github

The work was produced for 2012 Global Visualization Marathon by Liang Hai, Zhang Yixuan and me. Through analyzing 70,000 tweets on 2012 US presidential election using Natural Language Processing techniques, we visualized the evolutionary process of users’ discourse over 16 days before the election.

The tools we used include Processing, R and Phothoshop.
